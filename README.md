# XDownload

multi-thread, broken-point-resume ,pause download library for Android

# Usage

```java
repositories {
    maven {
        url "https://jitpack.io"
    }
}

dependencies {
    compile 'com.github.MasonLiuChn:XDownload:1.0.0'
}
```
